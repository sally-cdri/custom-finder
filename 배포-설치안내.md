# SallyFinder 설치 안내

내 맘대로 폴더를 만들고 링크·파일·이미지·텍스트를 한곳에서 관리하는 개인용 Finder 앱입니다. (Intel·Apple Silicon 공용 유니버설 빌드)

## 1. 설치
1. 받은 `SallyFinder_0.3.0_universal.dmg` 를 더블클릭
2. 열린 창에서 **SallyFinder 앱을 Applications 폴더로 드래그**

## 2. 첫 실행 (중요 — 서명 안 된 앱)
Apple 공증을 받지 않은 빌드라, 처음에 macOS가 실행을 막습니다. 아래 중 하나로 한 번만 풀어주면 됩니다.

**방법 A (권장)**
- `응용 프로그램`에서 **SallyFinder 아이콘 우클릭 → 열기** → 경고창에서 **"열기"** 클릭
- (그냥 더블클릭하면 막히지만, 우클릭→열기는 통과됩니다. 최초 1회만)

**방법 B — A로 안 되거나 "손상되었기 때문에 열 수 없습니다"가 뜰 때**
- 터미널에 아래 한 줄 실행:
  ```
  xattr -dr com.apple.quarantine /Applications/SallyFinder.app
  ```
- 그 후 앱 실행

## 3. 사용법
- **폴더**: `+ 추가 → 새 폴더`. 폴더를 더블클릭해 들어가고, 상단 경로(breadcrumb)·왼쪽 사이드바로 이동
- **항목 추가 (4가지)**
  - `+ 추가` 버튼: 폴더 / 링크 / 텍스트 메모 / 파일 선택 / 이미지 선택
  - Finder에서 파일·이미지를 **드래그&드롭** → 현재 폴더로 복사
  - `Cmd+V`: 클립보드의 이미지 / 텍스트 / URL 을 현재 폴더에 추가
- **열기 (더블클릭)**: 폴더 진입 · 링크는 브라우저 · 파일은 기본 앱 · 이미지는 미리보기 · 텍스트는 편집
- **정리**: 우클릭 메뉴(열기 / 이름 변경 / 삭제) · `Delete` 키 삭제 · 다른 폴더로 드래그해 이동 · 이름 더블클릭으로 이름 변경
- **검색**: 오른쪽 위 검색창에 입력 → 전체 폴더에서 이름·메모·링크로 찾기 (각 항목 경로 함께 표시)

## 참고
- 추가한 파일·이미지는 앱 저장소로 **복사**되어 보관됩니다. 원본을 옮기거나 지워도 영향 없습니다.
- 모든 데이터는 본인 맥에만 로컬 저장됩니다 (`~/Library/Application Support/com.sally.customfinder`).

## (개발자) 새 버전 배포
```bash
npm run release -- <x.y.z> "변경 내용 한 줄"
```
- 버전 올림 → 커밋 → 유니버설 `.dmg` 빌드까지 자동
- git 원격(origin)이 연결돼 있으면 push·태그·GitHub Release까지 진행
- 원격이 없으면 로컬 DMG 빌드만 하고, 연결 방법을 안내합니다:
  ```bash
  gh repo create sally-cdri/custom-finder --private --source=. --remote=origin --push
  ```

## (개발자) 사용자별 이름으로 빌드
사람마다 다른 앱 이름(JudyFinder, MaxFinder 등)으로 만들 수 있다. 이름별로 별개 앱이 되고(데이터 저장소도 독립), 한 맥에서 공존한다.
```bash
npm run build:named -- JudyFinder
```
- `productName`·창 제목·식별자(`com.sally.judyfinder`)를 빌드 동안만 임시 치환 → `JudyFinder.app` / `JudyFinder_<버전>_universal.dmg` 생성 (`tauri.conf.json`은 자동 원복)
- 같은 버전의 GitHub Release가 있으면 그 자산으로 자동 첨부
- 업데이트 시에는 이름마다 한 번씩 다시 실행해야 한다.
